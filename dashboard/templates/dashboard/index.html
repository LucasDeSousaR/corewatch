Bem-vindo ao CoreWatch

Sobre o Projeto

O CoreWatch é um sistema de monitoramento de LPARs em ambientes Mainframe. Ele permite visualizar o uso de CPU, MSU e SMF, facilitando a análise e o diagnóstico de desempenho.

Projeto de Conclusão de Curso de Análise e Desenvolvimento de Sistemas

FATEC Campinas

Este projeto foi desenvolvido como parte de um trabalho acadêmico da instituição FATEC Campinas Nesse projeto foi utilizado o framework Django para o desenvolvimento do sistema de monitoramento e a plataforma ZOWE para desenvolver dentro do mainframe. Banco de dado utilizado foi o MongoDB nos proporcionando uma estrutura flexível e escalável para armazenar os dados de monitoramento.
